# Prompt Optimize（レガシーシム）

`/prompt-optimize` をまだ呼び出している場合のみ使用してください。メンテナンスされているワークフローは `skills/prompt-optimizer/SKILL.md` にあります。

## 正規サーフェス

- `prompt-optimizer` スキルを直接使用することを推奨します。
- このファイルは互換性のエントリーポイントとしてのみ保持してください。

## 引数

`$ARGUMENTS`

## 委譲

`prompt-optimizer` スキルを適用します。
- アドバイスのみ: プロンプトを最適化し、タスクを実行しません。
- 推奨ECCコンポーネントとすぐに実行可能なプロンプトを返します。
- ユーザーが実際に直接実行を望んでいる場合は、シム内にとどまるのではなく通常のタスクリクエストを行うよう伝えてください。
