# Quality Gateコマンド

ファイルまたはプロジェクトスコープに対してECC品質パイプラインをオンデマンドで実行します。

## 使用方法

`/quality-gate [path|.] [--fix] [--strict]`

- デフォルトターゲット: 現在のディレクトリ（`.`）
- `--fix`: 設定されている場合に自動フォーマット/修正を許可
- `--strict`: サポートされている場合に警告でも失敗

## パイプライン

1. ターゲットの言語/ツーリングを検出。
2. フォーマッターチェックを実行。
3. 利用可能な場合にlint/型チェックを実行。
4. 簡潔な修正リストを生成。

## 注意事項

このコマンドはフック動作をミラーしますが、オペレーターが呼び出すものです。

## 引数

$ARGUMENTS:
- `[path|.]` オプションのターゲットパス
- `--fix` オプション
- `--strict` オプション
