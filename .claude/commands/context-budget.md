# Context Budgetオプティマイザー（レガシーシム）

`/context-budget` をまだ呼び出している場合のみ使用してください。メンテナンスされているワークフローは `skills/context-budget/SKILL.md` にあります。

## 正規サーフェス

- `context-budget` スキルを直接使用することを推奨します。
- このファイルは互換性のエントリーポイントとしてのみ保持してください。

## 引数

$ARGUMENTS

## 委譲

`context-budget` スキルを適用します。
- ユーザーが指定した場合は `--verbose` を渡します。
- ユーザーが別途指定しない限り、200Kコンテキストウィンドウを想定します。
- ここでスキャンを再実装する代わりに、スキルのインベントリ、問題検出、優先順位付きの削減レポートを返します。
